## My-Blog

# 开发中！！！！

`my-blog` 主要使用了 `react`、 `react-redux` 、`koa2`、`mongose` 

 目前正在做响应式，与完善功能....

### npm command

```
# install dependencies
npm install

# 开发,跑在本地8080端口
npm run start-dev

# webpack 打包
npm run build

# 部署,也是8080端口
npm run start-pro

# forever 运行
npm run forever-start
# forever 停止运行
npm run forever-stop

```
