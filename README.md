## My-Blog

# 开发中！！！！

`my-blog` 主要使用了 `react`、`koa2`、`mongose` 等

正在渐渐用 react-redux 重构中...

### npm command

```
# install dependencies
npm install

# 开发,跑在本地8080端口
npm run start-dev

# webpack 打包
npm run build

# 部署,也是8080端口
npm run start-pro

# forever运行
npm run forever-start
npm run forever-stop

```
