## My-Blog

### 技术栈

`react`、 `react-redux` 、`koa2`、`mongose` 

### npm command

```
# install dependencies
npm install

# 开发,跑在本地8080端口
npm run start-dev

# webpack 打包
npm run build

# 部署,也是8080端口
npm run start-pro

# forever 运行
npm run forever-start

# forever 停止运行
npm run forever-stop

```
# open mongodb
/usr/local/mongodb/bin/mongod --config mongo.conf
